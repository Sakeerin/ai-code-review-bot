## What is AI Review Bot?

AI Review Bot is a modern, AI-native code review tool that integrates with GitHub and GitLab. It uses Claude to analyze your Pull Requests and Merge Requests, providing framework-aware feedback and suggestions.

## Key Features

- **Framework-aware**: Built-in profiles for Laravel, Vue, and TypeScript.
- **Customizable**: Define your own rules using `.reviewbot.yml`.
- **Fast**: Built on Cloudflare Workers for 0ms cold starts.
- **Detailed**: Inline comments with severity levels and suggestions.

## Getting Started

1. **Install the App**: Connect your GitHub or GitLab account.
2. **Configure**: Add a `.reviewbot.yml` file to your repository (optional).
3. **Open a PR**: Let the AI analyze your code and provide feedback.
